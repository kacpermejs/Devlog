My thesis project - Design and implementation of a real-time strategy game using the Unity engine

Official abstract:
The main goal of this thesis was to design and implement a strategy game, containing a system for collecting and processing in-game resources. The game belongs to the real-time strategy genre. The implementation includes the use of the A* algorithm to find a path in a graph representing the game world. This algorithm is used by artificial intelligence units. In the implementation, efforts were made to adhere to the principles of the object-oriented programming paradigm and related software design patterns. The final result of the work is a game prototype. In the prototype there is a small number of objects only for the purpose of testing all the features. The designed system is suitable for easy introduction of new environment elements and thus expanding the game contents, without the need of interfering with the game code.
